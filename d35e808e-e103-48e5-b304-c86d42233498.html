VZX Creative


The VZX music visualizer family of programs generates awesome audio reactive graphics in real time from any audio source - music player, youtube, microphone, line-in etc.

VZX Creative is an application suite for creatives and professionals who wish to have better control over live visualization playback and or create their own music visualizations.

VZX Creative is comprised of 2 separate products:

VZX Player Professional

  • Plays Vovoid's full range of DLC visuals in addition to custom made visuals.
    • Plays DLC you have already bought for VZX Player (Consumer player) but not with all features
    • Plays Pro Player versions of DLC
  • Instead of using "game capture", supports SPOUT output for low-latency integration with other software (OBS can support SPOUT input with a plugin).

  • Output separated from GUI:
    • The Output (VZX Pro Player) is a separate application/binary from the GUI (VZX Pro Controller).
    • Run GUI/Controller on one monitor, graphics output on another monitor.
    • Run GUI/Controller on one computer, output on another computer.
    • Control commands are sent over the network - locally or over the LAN.
  • Audio
    • Audio recording is also detached into its own application, can also run on a separate computer
    • Audio data is also sent via network - locally or over the LAN.
  • Preset creation with:
    • Support for tweak parameters (all DLC including consumer DLC packs)
    • Up to 4 layers of custom overlay images with full industry standard blend mode support (Only if you own Pro DLC)
  • Color Palette manager which can be used in tweak/overlay parameters in presets and chosen during live playback
  • Playlist support, multiple playlists
  • Handling of multiple complete states as mostly human-readable JSON files (where one state includes presets, palettes playlists etc)
  • In-app documentation
  • UI supports click actions on touch screens - for instance using an android tablet as an extra screen
  • Exactly position the output window for streaming setups

VZX Artiste

  • SPOUT with alpha support for designing custom GPU-accelerated stream overlays
  • Modular programming environment focused on letting you control and tweak rendering of music visuals and other content
  • Immersive UI focused on fine-tuning the graphical output
  • Over 200 programming modules covering maths, rendering, audio input, image loading and more
  • In-app and online reference documentation
